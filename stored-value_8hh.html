enum mutation_type_t

Mutation types as returned by store commands.

Enumerator:
INVALID_VBUCKET 

Storage was attempted on a vbucket not managed by this node.

NOT_FOUND 

The item was not found for update.

INVALID_CAS 

The wrong CAS identifier was sent for a CAS update.

WAS_CLEAN 

The item was clean before this mutation.

WAS_DIRTY 

This item was already dirty before this mutation.

IS_LOCKED 

The item is locked and can't be updated.

NOMEM 

Insufficient memory to store this item.

SUCCESS 

Stoarage was successful (see also WAS_CLEAN/WAS_DIRTY).
